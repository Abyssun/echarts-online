# 数据可视化概念

## 什么是数据可视化

- 把数据以更直观的形式展现      ---> 图表

## 数据可视化的好处

- 清晰有效地传达与沟通信息

- 观察隐藏在数据中的信息

## 可视化的实现方式

-  报表类（主要面向非技术人员）
   -  Excel
   -  水晶报表


- 商业智能BI
  - Microsoft BI
  - Power BI


- 编码类
  - Echarts.js
  - D3.js
